Bordes CSS

Estilos de borde

La propiedad border-style especifica qué tipo de borde se mostrará.

Se permiten los siguientes valores:

  • dotted - Define un borde de puntos
  • dashed - Define un borde discontinuo
  • solid - Define un borde sólido
  • double - Define un borde doble
  • groove - Define un borde acanalado 3D (depende de border-color)
  • ridge - Define un borde estriado 3D (depende de border-color)
  • inset - Define un borde interior 3D
  • outset - Define un borde exterior 3D
  • none - Sin borde
  • hidden - Borde oculto

La propiedad border-style puede tener de uno a cuatro valores (superior, derecho, inferior e izquierdo).

Ejemplo

Código

<p class="dotted">Borde punteado.</p>
<p class="dashed">Borde con guiones.</p>
<p class="solid">Línea sólida.</p>
<p class="double">Doble línea.</p>
<p class="groove">Surco.</p>
<p class="ridge">Rugoso.</p>
<p class="inset">Borde interior.</p>
<p class="outset">Borde exterior.</p>
<p class="none">Sin borde.</p>
<p class="hidden">Borde oculto.</p>
<p class="mix">Borde mixto.</p>

Ancho del borde

La propiedad border-width especifica la anchura de los cuatro bordes.

Puede definirse en px, pt, cm, em, etc., o mediante los valores predefinidos: fino, medio o grueso.

Color del borde

La propiedad border-color se utiliza para establecer el color de los cuatro bordes.

  • Nombre - Ej.: "rojo"
  • Hex - Ej.: "#FF0000"
  • RGB - Ej.: "rgb(255,0,0)"
  • transparente

Border - Lados individuales

Es posible especificar un estilo de borde distinto para cada lado.

Border - Abreviación

La propiedad border permite definir border-width, border-style y border-color en una sola línea.

Bordes redondeados

La propiedad border-radius se utiliza para añadir bordes redondeados.

Ejemplo

<p class="normal">Borde normal</p>
<p class="round1">Redondeado 1</p>
<p class="round2">Redondeado 2</p>
<p class="round3">Redondeado 3</p>

Nota: La propiedad "border-radius" no está soportada por IE8 ni versiones anteriores.
