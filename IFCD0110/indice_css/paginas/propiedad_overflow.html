Propiedad Overflow

Overflow: visible

Este texto se desborda fuera del contenedor porque overflow está configurado como "visible". No se oculta nada y el contenido puede sobresalir libremente.

Overflow: hidden

Con overflow: hidden, todo el contenido que se salga del tamaño del contenedor queda oculto. No aparecen barras de desplazamiento.

Overflow: scroll

Con overflow: scroll siempre aparecen barras de desplazamiento, incluso si no son necesarias. El usuario puede mover el contenido tanto vertical como horizontalmente.

Overflow: auto

Con overflow: auto solo aparecen barras de desplazamiento cuando el contenido realmente lo necesita. Es la opción más usada en diseño moderno.

Overflow-x y Overflow-y

Aquí se aplica overflow-x: scroll y overflow-y: hidden. Esto provoca que solo haya desplazamiento horizontal, útil para tablas o listas largas.
