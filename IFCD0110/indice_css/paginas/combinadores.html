Pseudo-elementos CSS

Un pseudo-elemento CSS se utiliza para aplicar estilo a partes específicas de un elemento. Por ejemplo:

  • Estilizar la primera letra o la primera línea de un elemento.
  • Insertar contenido antes o después del contenido real.
