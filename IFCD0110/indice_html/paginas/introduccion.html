¿Qué es HTML?

HTML es el lenguaje de marcado estándar para la creación de páginas Web.

  • HTML significa Hyper Text Markup Language.
  • HTML describe la estructura de las páginas Web usando marcado.
  • Los elementos HTML son los bloques de construcción de páginas HTML.
  • Los elementos HTML están representados por etiquetas.
  • Las etiquetas HTML están ordenadas en niveles como <head>, <p>, <table>...
  • Los navegadores no muestran las etiquetas HTML, pero las usan para representar el contenido.

Estructura básica de un documento HTML

Ejemplo

<!DOCTYPE html>
<html lang="es-ES">
<head>
  <meta charset="utf-8">
  <title>Título de la página</title>
</head>
<body>
  <h1>Mi primer encabezado.</h1>
  <p>Mi primer párrafo.</p>
</body>
</html>

Explicación de las etiquetas básicas

  1. La etiqueta <!DOCTYPE html> define el tipo de documento.
  2. La etiqueta <html lang="es-ES"> define el elemento raíz.
  3. La etiqueta <head> define los metadatos.
  4. La etiqueta <meta charset="utf-8"> define el juego de caracteres.
  5. La etiqueta <title> especifica el título.
  6. La etiqueta <body> contiene el contenido visible.
  7. La etiqueta <h1> define el encabezado principal.
  8. La etiqueta <p> define un párrafo.

Etiquetas HTML

Las etiquetas HTML son los nombres de elementos entre paréntesis angulares:

<Nombre de etiqueta> contenido ... </nombre de la etiqueta>

Etiquetas HTML
Normalmente vienen en pares como <p> y </p>
La primera etiqueta es la etiqueta de inicio y la segunda la etiqueta de fin
La etiqueta de fin incluye una barra inclinada antes del nombre

Nota: La etiqueta de inicio también se llama etiqueta de apertura y la final etiqueta de cierre.

Estructura de una página web

A continuación se muestra una visualización de la estructura HTML:

<html>
<head>
<title>Título de la página</title>
</head>
<body>
<h1>Esto es un encabezado.</h1>
<p>Esto es un párrafo.</p>
<p>Esto es otro párrafo.</p>
</body>

Nota: Sólo el contenido dentro de <body> se muestra en el navegador.
