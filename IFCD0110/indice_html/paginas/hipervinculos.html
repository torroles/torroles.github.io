enlace en imagen 
Para crear un vínculo que llame a una parte de la página usamos:

<id="nombre"> → donde queremos ir
<a href="#nombre"> → enlace que nos lleva
            
Podemos cambiar los colores de los enlaces usando CSS:

A:LINK     → enlace sin visitar
A:ACTIVE   → cuando pulsas
A:VISITED  → ya visitado
A:HOVER    → cuando pasas el ratón

Ejemplo:

a:link {
    color: negro;
    background: none;
    text-decoration: none;
}

a:hover {
    color: verde;
}
buscador

Para poner un enlace usamos:

<a href="https://www.google.es/">buscador</a>

También podemos ponerlo dentro de un texto:
Para buscar, el mejor buscador es GOOGLE.

Después de la dirección podemos usar el atributo REL:
SPONSORED → enlace de pago
UGC → generado por usuario
NOFOLLOW → los buscadores no lo siguen

El atributo TARGET:

_SELF → misma pestaña
_BLANK → nueva pestaña
_PARENT → marco padre
_TOP → cuerpo completo

Direcciones absolutas vs relativas:

Absolutas → incluyen https://www
Relativas → pertenecen a la misma página

Para poner un enlace dentro de una imagen:
Se coloca la etiqueta IMG dentro de A.

Para crear un botón con enlace:

<button onclick="document.location='direccion'">nombre</button>

Con la etiqueta A podemos usar TITLE para mostrar información al pasar el ratón.
