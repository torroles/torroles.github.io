Etiquetas que definen una Lista

Etiqueta Descripción
<ul> Define una lista desordenada
<ol> Define una lista ordenada
<li> Define los ítems de la lista
<dl> Define una lista de definición
<dt> Define un término de la lista de definición
<dd> Define la descripción del término

Nota: Los elementos de lista pueden contener nuevas listas u otros elementos HTML.

Listas desordenadas (ul)

La etiqueta <ul> define una lista desordenada. Utiliza <ul> junto con <li>.

Atributo Valor Descripción
compact compact No soportado por HTML5
type
  1. disc
  2. square
  3. circle
 No soportado por HTML5

Listas ordenadas (ol)

La etiqueta <ol> define una lista ordenada. Puede ser numérica o alfabética.

  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk

Listas de definición (dl)

La etiqueta <dl> define una lista de definición. Se usa junto con <dt> y <dd>.

Coffee
Black hot drink
Vegetal origin
Milk
White cold drink
Animal origin
