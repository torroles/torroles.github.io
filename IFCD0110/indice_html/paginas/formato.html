El formato del texto

El formato de texto es el marcado que se aplica a un texto para añadirle datos de forma.

Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido.

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de las páginas web.

Los usuarios hojean las páginas por sus títulos. Es importante utilizar encabezados para mostrar la estructura del documento.

Los encabezamientos <h1> deben usarse para las partes principales, seguidos por <h2>, <h3>…

Nota: Usa encabezados solo para encabezados, no para hacer texto grande o en negrita.

Etiquetas que dan formato al texto

HTML define elementos especiales para dar significado al texto.

HTML utiliza elementos como <b> y <i> para formato visual.

Etiqueta Descripción Ejemplo Utilización
<b> Texto en negrita Este texto está en negrita <b>Texto</b>
<i> Texto en cursiva Este texto está en cursiva <i>Texto</i>
<strong> Texto importante Texto importante <strong>Texto</strong>
<em> Texto enfatizado Texto enfatizado <em>Texto</em>
<small> Texto pequeño Texto pequeño <small>Texto</small>
<sup> Superíndice Texto superíndice <sup>Texto</sup>
<sub> Subíndice Texto subíndice <sub>Texto</sub>
<ins> Texto insertado Texto insertado <ins>Texto</ins>
<u> Subrayado Texto subrayado <u>Texto</u>
<s> Tachado Texto tachado <s>Texto</s>
<del> Texto eliminado Texto eliminado <del>Texto</del>
<mark> Texto marcado Texto marcado <mark>Texto</mark>

Nota: <strong> y <em> tienen significado semántico, no solo visual.

Etiquetas que separan el texto

<p> separa texto añadiendo espacio antes y después.

<br> inserta un salto de línea sin iniciar un nuevo párrafo.

<pre> conserva espacios y saltos de línea.

Etiqueta Descripción Visualización Utilización

Esto es un párrafo

<pre> Texto preformateado

Nota: Usa <pre> para mostrar código o formato especial.
