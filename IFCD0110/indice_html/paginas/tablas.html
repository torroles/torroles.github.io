Etiquetas que definen una tabla

Una tabla HTML se define con la <table> etiqueta.

Cada fila de la tabla se define con la etiqueta <tr>. Un encabezado de la tabla se define con la etiqueta <th>. Por defecto, los encabezados están en negrita y centrados. Las celdas de datos se definen con la etiqueta <td>.

Etiqueta Descripción
<table> Define una tabla
<tr> Define una fila
<th> Define el encabezado de la tabla
<td> Define una celda dentro de una fila
<caption> Define el título o leyenda de la tabla
<colgroup> Especifica un grupo de columnas para dar formato
<col> Especifica propiedades de columna
<thead> Agrupa el contenido del encabezado
<tbody> Agrupa el contenido del cuerpo
<tfoot> Agrupa el contenido del pie

El atributo Scope

El atributo scope especifica si una celda de encabezado es una cabecera de una columna, fila o grupo de columnas o filas. No tiene ningún efecto visual en los navegadores web normales, pero puede ser utilizado por los lectores de pantalla.

Los atributos Colspan y Rowspan

El atributo colspan define el número de columnas que una celda debe ocupar. El atributo rowspan especifica el número de filas que una celda debe ocupar.

Los atributos Colgroup y Col

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para el formato. Es útil para aplicar estilos a columnas completas sin repetir estilos en cada celda.

La etiqueta <col> especifica propiedades de columna dentro de un <colgroup>. También permite aplicar estilos a columnas completas.
