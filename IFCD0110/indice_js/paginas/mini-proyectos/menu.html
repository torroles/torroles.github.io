Menú con capas y submenús

Objetivo: Implementar un sistema de navegación mediante menús colapsables interactivos. El componente gestiona de manera asíncrona los estados visuales del árbol de enlaces mediante eventos de puntero (`onmouseover` y `onmouseout`), delegando el comportamiento animado a transiciones nativas de CSS sobre la propiedad `height` para optimizar el rendimiento de renderizado.
