El Juego de la Serpiente (Snake)

Objetivo: Desarrollar el clásico videojuego arcade 'Snake' mediante manipulación del elemento Canvas de HTML5. El script implementa un bucle de juego asíncrono que refresca el estado físico del juego, gestiona una estructura de datos de matriz para el cuerpo de la serpiente, calcula colisiones perimetrales/autofagia y expone un sistema de control adaptativo (teclado para escritorio y API táctil mediante botones en pantalla para dispositivos móviles).
