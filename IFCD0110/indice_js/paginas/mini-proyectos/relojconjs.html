Reloj Digital Dinámico

Objetivo: Construir un cronómetro digital síncrono con el sistema del cliente en tiempo real. El script instancia de forma cíclica el objeto nativo `Date` de JavaScript, procesa y normaliza los componentes de tiempo (horas, minutos y segundos) aplicando formateo de cadenas nativo (`padStart`) y gestiona la reactivación asíncrona del hilo de ejecución mediante la API `setTimeout` cada 1000 milisegundos.
