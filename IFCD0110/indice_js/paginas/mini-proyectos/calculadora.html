Calculadora Simple

Objetivo: Crear un motor de cálculo aritmético básico mediante la manipulación del DOM. El script valida la entrada de datos numéricos flotantes, implementa una estructura condicional indexada para las cuatro operaciones básicas y cuenta con control de excepciones para prevenir la división por cero de manera asíncrona.

Introduce los valores y selecciona una operación.
