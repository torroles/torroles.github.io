Formularios de JavaScript

Ejercicio 1: Validación de formularios en JavaScript

Objetivo: Evaluar si un campo obligatorio se encuentra vacío de forma manual antes del envío. Si la validación falla, se detiene el flujo del formulario lanzando una alerta.

Ejercicio 2: Validación de Entrada Numérica

Objetivo: Controlar en tiempo de ejecución que el valor introducido por el usuario sea estrictamente un operando numérico comprendido entre 1 y 10.

Introduce un número para validar.

Ejercicio 3: Validación automática de formularios HTML

Objetivo: Delegar la validación primaria de restricciones en el motor nativo del navegador mediante el atributo required.

Ejercicio 4: Validación de restricciones HTML5

HTML5 introduce el concepto de validación de restricciones, estructurado en base a atributos nativos y pseudoselectores CSS (:invalid, :valid).

AtributoDescripción
disabledDeshabilita el control, impidiendo edición y envío.
max / minLímites numéricos o cronológicos.
patternDefine una RegEx para contrastar el valor.
requiredCampo obligatorio para procesar la estructura.
