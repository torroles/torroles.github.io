Cronómetro de Sesión

Objetivo Técnico: Uso de los métodos setInterval() y clearInterval() del objeto window para gestionar la ejecución de código en intervalos de tiempo determinados.

Tiempo transcurrido: 0 segundos

let segundos = 0; let interval; function iniciar() { if(!interval) interval = setInterval(() => { segundos++; document.getElementById('contador').textContent = segundos; }, 1000); } function parar() { clearInterval(interval); interval = null; }
