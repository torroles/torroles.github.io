Algoritmo de Control de Identidad (Letra DNI)

Objetivo: Generar la clave alfabética de validación de un DNI español. El proceso aplica la operación aritmética de módulo 23 al operando numérico introducido, utilizando el resto resultante como puntero indexado sobre la cadena maestra estandarizada de asignación.

Introduce los dígitos numéricos para verificar la letra de control.
