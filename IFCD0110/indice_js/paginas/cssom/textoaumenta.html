Editor de Animación de Fuentes

Objetivo Técnico (CSSOM): Implementación de una animación controlada mediante el cálculo recursivo de estilos. A diferencia de las transiciones CSS puras, este script manipula directamente la propiedad style.fontSize en el objeto CSSOM, permitiendo una lógica de control avanzada (pausa, reinicio y velocidad variable) gestionada mediante el bucle de eventos de JavaScript y setTimeout.
